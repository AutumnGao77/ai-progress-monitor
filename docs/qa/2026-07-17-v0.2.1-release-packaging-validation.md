# v0.2.1 发布包验包记录

## 结论

v0.2.1 的 macOS 与 portable 两份发布包均已通过自动化构建和独立验包，可以进入代码审查与 PR 阶段。macOS 包只提供一个正式 App，不再让用户在普通版与 Floating 版之间选择；portable 包单独承载命令行集成、诊断脚本和 Windows 轻量预览。

当前 macOS App 是 **macOS 13+、Apple Silicon arm64、ad-hoc 签名、未公证** 的本地构建。它不应被描述成已完成 Apple Developer ID 签名或 notarization 的正式分发包。GitHub Release 上传后，仍需从 Release 页面重新下载一次并人工验证 Gatekeeper 与首次启动体验。
