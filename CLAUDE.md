# Claude Code 指南

## 项目

**AI 工作进度监控助手**：本地优先、低侵入的 AI 工作进度监控桌面助手，聚合 Claude Code、Codex 及后续可配置 AI 工具在终端和桌面端中的状态，并在需要时提醒用户回到对应窗口。

> 完整项目说明、运行命令、测试规范、提交流程等，见 [AGENTS.md](./AGENTS.md)。

## 目录速查

- `src/` - 应用代码
- `tests/` - 测试
- `scripts/` - 启动/辅助脚本
- `native/` - macOS / Windows 原生外壳
- `docs/` - PRD / QA / 发布清单等正式文档
- `notes/` - 草稿、临时笔记

## 协作规则

- 项目主文档以中文为准；公开入口可同时维护英文版，例如 `README.en.md`
- 不自动提交 / 不自动推送，除非明确要求
- 敏感信息（凭据、个人姓名）不进仓库；本机姓名用 `Gao` 占位
- Git 关键操作（commit / push / rebase / amend 等）前先用通俗中文解释每步作用与风险
- 改动完成后，若涉及用户可见 UI，必须在本机实测后再报告完成
